## Getting started

### Requirement
`Dart >=2.17.0 < 3.0.0` and `Flutter >=1.17.0`

## Usage

This is how you can call the chatbox and use the UI.

```dart
const ChatBox(
message: "Text message here",
recieved: false,
messageColor: Colors.greenAccent,
textColor: Colors.black,//optional
);
```
